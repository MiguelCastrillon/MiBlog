Tutorial Completo: Cómo Utilizar Git

Git es una herramienta esencial para el control de versiones en desarrollo de software. Aquí te ofrecemos un tutorial completo para aprender a utilizar Git, desde la inicialización de un repositorio local hasta la creación de ramas y la conexión a un repositorio remoto.

1. Inicialización del repositorio local

Para empezar a usar Git, primero necesitas inicializar un repositorio local en tu máquina con el siguiente comando:

git init

Este comando crea un repositorio vacío en tu directorio actual, permitiéndote rastrear cambios en tus archivos. Usa el comando git status para verificar el estado de tu repositorio.

2. Creación de branches

Las ramas te permiten trabajar en nuevas funcionalidades sin afectar la rama principal. Crea una nueva rama con este comando:

git checkout -b nombre-de-la-rama

Esto creará y te moverá automáticamente a la nueva rama. Para fusionar esta rama con la principal, usa los siguientes comandos:

git checkout main
git merge nombre-de-la-rama

3. Agregar un repositorio remoto

Una vez configurado tu repositorio local, es momento de conectarlo a un repositorio remoto, como GitHub. Usa el siguiente comando para agregar el remoto:

git remote add origin https://github.com/tu-usuario/tu-repositorio.git

Verifica la conexión con el comando:

git remote -v

Para subir los cambios a tu repositorio remoto, utiliza:

git add nombre-del-archivo o . para todos los archivos
git commit -m "Mensaje"
git push -u origin main

Conclusión

Este tutorial cubre los comandos básicos para utilizar Git, incluyendo la inicialización de un repositorio, la creación de ramas, y la conexión a un repositorio remoto. Git es una herramienta fundamental para cualquier desarrollador que quiera gestionar su código de manera eficiente.
